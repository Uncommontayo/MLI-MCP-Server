# Example Prompts for MCP Gmail Server

Use these example prompts to test the Gmail MCP server with Claude Desktop.

## Reading Unread Emails

### Basic Email Check
```
Can you check my unread emails?
```

### Limited Results
```
Show me my 5 most recent unread emails
```

### Detailed Summary
```
Check my unread emails and give me a summary of each one, including who sent it and what it's about
```

## Creating Draft Replies

### Simple Reply
```
Draft a reply to the email from [sender name] saying that I'll get back to them tomorrow
```

### Professional Response
```
Create a professional draft reply to the meeting invitation from Sarah, 
confirming my attendance for Tuesday at 2pm
```

### Specific Content
```
Draft a reply to the email about the project proposal. 
Say that I've reviewed it and have a few questions:
1. What's the timeline?
2. What's the budget?
3. Who else is on the team?
```

## Combined Workflows

### Triage and Respond
```
1. Check my unread emails
2. Identify which ones need immediate responses
3. Draft polite acknowledgment replies for urgent ones
```

### Batch Processing
```
Check all my unread emails from today and draft brief acknowledgment 
replies for each one, letting them know I'll respond in detail soon
```

### Smart Filtering
```
Check my unread emails and:
1. Show me any that mention "urgent" or "deadline"
2. Draft appropriate responses based on the content
```

## Advanced Use Cases

### Context-Aware Replies
```
Read the email thread from [person] and draft a reply that:
- Acknowledges their concerns
- Provides a timeline for resolution
- Maintains a professional but friendly tone
```

### Multi-step Workflow
```
1. Find unread emails from this week
2. Categorize them by topic
3. Draft replies for the work-related ones
```

## Tips for Best Results

- **Be specific** about which email you're referring to (sender name, subject, etc.)
- **Provide context** for the reply you want drafted
- **Specify tone** if you want formal, casual, or specific style
- **Include details** that should be in the reply
- Review drafts in Gmail before sending!
